⚙ Settings

Account

Run claude setup-token locally and paste the sk-ant-oat… value. Lasts ~1 year, needs no refresh, and doesn't clash with your local login. Set once, forget. Required to create or start a session.

Change password

Your login password for this app. At least 8 characters.

Notifications

Get a push when Claude finishes a turn or needs your input — even with the app closed. While you're looking at the session it just chimes instead of buzzing.

On iPhone, notifications work only from the Home-Screen app: tap Share → Add to Home Screen, open ccchat from that icon, then turn this on.

Subscriptions are per-device — enable this on each phone/computer you want to be notified on.

Appearance

Pick a theme. Applies instantly and is remembered on this device.

Terminal

How the terminal opens from ⋯ → Terminal.

Optional mounts

Host paths (on the VPS) users can attach to a session. The description is injected into the session's system prompt. Admin-only mounts are selectable by admins only. Path aliases, expanded per session: @workspace = all users' workspaces root, @user = this user's folder — e.g. @workspace/@user mounts the user's own cross-chat folder.
